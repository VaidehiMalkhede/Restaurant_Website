What website is about ?

This website is about Restaurant Food Ordering Concept. It consists of navigation menu which includes Home, About, Menu, Gallary, and Order. Home sesction contains basic information. About section displays information about this project. Menu section contains different types of food menus along with its price with order button. Gallary section contains food images and information about the food along with order button. Order section contains the name, address, email, order no, your order etc and an order button which will redirect to order successfull page. And last is Team section which is about project members.
